# Vuetify.js Webpack SSR Template

## Build Setup

``` bash
vue-init vuetifyjs/webpack-ssr

cd webpack-ssr

# npm
npm install

# yarn
yarn
```

For additional information, please visit the [Official Documentation](https://vuetifyjs.com).
